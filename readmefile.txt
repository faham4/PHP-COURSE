# Weather App

This is a simple Weather App built using PHP. It allows users to check the current weather of any city by entering the city name. The app fetches weather data using the OpenWeatherMap API.

## Features

- Fetches weather data from OpenWeatherMap API.
- Displays the weather description and temperature in Celsius.
- Simple and responsive user interface.

## Requirements

- PHP 5.6 or later
- OpenWeatherMap API Key (free account)
